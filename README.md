# Privacy Policy - Masuyama Auto Lister

Last updated: January 31, 2026

## Overview

Masuyama Auto Lister is a personal desktop application for managing product listings.

## Data Collection

This application:

- Does NOT collect personal data from users
- Does NOT share data with third parties
- Only accesses Pinterest API to post pins on behalf of the authenticated user

## Data Storage

- API credentials are stored locally on your device only
- No data is transmitted to external servers other than official Pinterest APIs
